Contents


1) streets.py - Look at street endings in original file (‘nashville.osm’) and get frequency count for each. Exports results to “streets.csv” file.


2) zip_codes.py - Similar to streets.py, this gets a frequency count for each unique zip code entry and exports results to ‘zip_codes.csv’


3) audit_streets.py - Checks street endings and maps corrections where needed. Exports results to ‘nashville_audit.osm’

4) process_map.py - Takes audited file (‘nashville_audit.osm’) and puts it in json format. The file then exports to ‘nashvile_mongo.json.’


5) mongo_queries - Various queries used in mongoDB to get information on dataset.
